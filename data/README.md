# 📊 Dataset

## Téléchargement

Télécharge le dataset depuis Kaggle :

https://www.kaggle.com/datasets/snap/amazon-fine-food-reviews

Place les fichiers suivants ici :
- `Reviews.csv` (dataset original)

## Structure

Le dataset contient les colonnes suivantes :
- **Id** : Identifiant unique
- **ProductId** : ID du produit
- **UserId** : ID de l'utilisateur
- **ProfileName** : Nom du profil
- **HelpfulnessNumerator** : Nombre de votes utiles
- **HelpfulnessDenominator** : Nombre total de votes
- **Score** : Note de 1 à 5
- **Time** : Timestamp
- **Summary** : Résumé de l'avis
- **Text** : Texte complet de l'avis

## Note

Les fichiers CSV ne sont pas inclus dans le repository Git (trop volumineux).
